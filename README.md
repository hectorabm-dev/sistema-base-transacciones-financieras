# Sistema Base de Transacciones Financieras

Programa desarrollado en Python que permite gestionar transacciones financieras básicas mediante el registro de depósitos y retiros.

## 📌 Funcionalidades

* Registro de depósitos (valores positivos)
* Registro de retiros (valores negativos)
* Cálculo de balance total
* Cálculo de promedio de transacciones
* Conteo de depósitos y retiros
* Identificación del retiro de mayor monto
* Análisis de retiros mayores a $500

## ⚙️ Uso

1. Ejecuta el programa en Python
2. Selecciona una opción del menú:

   * 1 → Depositar
   * 2 → Retirar
   * 3 → Salir
3. Ingresa los montos según la operación
4. Al salir, el sistema mostrará el resumen de transacciones

## 🧠 Descripción técnica

El sistema utiliza estructuras básicas de programación en Python como:

* Listas para almacenamiento de datos
* Bucles `while` para interacción continua
* Condicionales para control de flujo
* Ciclos `for` para procesamiento de datos
